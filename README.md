# generator-angular-rockr

> this project is a fork from [gulp angular generator](https://github.com/viniciusdacal/generator-angular-rockr)
> Yeoman generator for AngularJS + Gulp.

> Lets you quickly set up a project with:
> * your favorite technologies
> * web best pratices.

> Gulp provide fast workspace with quick feedback.

## Usage

More informations, options, parameters in the [usage documentation page](docs/usage.md)

### Install

##### Install required tools `yo`, `gulp` and `bower`:
```
npm install -g yo gulp bower
```

##### Install `generator-angular-rockr`:
```
npm install -g generator-angular-rockr
```

### Run

##### Create a new directory, and go into:
```
mkdir my-new-project && cd $_
```

##### Run `yo gulp-angular`, and select desired technologies:
```
yo gulp-angular
```
